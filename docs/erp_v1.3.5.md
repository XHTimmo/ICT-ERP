# ERP 系统 (ERP System) 技术文档

**版本**: v1.3.5  
**最后更新**: 2026-04-02

## 1. 项目简介

本 ERP 系统是一个基于 Electron + Vue 3 的桌面应用程序，聚焦「报销管理」。
v1.3.5 版本主要优化了报销列表的分组展示与分页体验，并修复了批量导出时的文件路径兼容问题。

核心能力：
- **分组展示**：报销列表按状态分组展示，信息分区更清晰。
- **灵活筛选**：支持快速隐藏“已完成”记录，聚焦待处理事项。
- **分页优化**：支持每页 10/20 条切换，分组内独立分页。
- **导出稳定**：批量导出补齐相对路径转绝对路径与文件存在性校验。
- **凭证查看**：支持在凭证列表中直接打开文件。

## 2. 技术栈 (Tech Stack)

- **桌面框架**: Electron (主进程/渲染进程架构)
- **前端框架**: Vue 3 (Composition API) + Vite
- **UI 组件库**: Element Plus
- **可视化**: ECharts
- **数据存储**: Better-SQLite3 + Electron-Store
- **自动更新**: electron-updater
- **CI/CD**: GitHub Actions

## 3. v1.3.5 关键更新

### 3.1 报销列表交互升级
- **状态分组卡片**：列表由单表格调整为按状态分组展示，提升浏览效率。
- **分组内分页**：每个状态分组支持独立分页与页码记忆。
- **隐藏已完成**：新增“显示已完成 / 隐藏已完成”开关，便于日常跟进未完成事项。
- **状态筛选动态化**：状态筛选项由固定写死调整为基于状态配置渲染。

### 3.2 导出链路修复增强
- **路径兼容**：导出时对凭证路径做绝对路径转换，兼容历史相对路径数据。
- **文件校验**：仅在文件存在时写入压缩包，避免因单个缺失文件导致整体失败。
- **打开导出结果**：导出成功后支持直接打开文件所在路径。
