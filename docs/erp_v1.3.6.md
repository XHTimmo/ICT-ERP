# ERP 系统 (ERP System) 技术文档

**版本**: v1.3.6  
**最后更新**: 2026-04-03

## 1. 项目简介

本 ERP 系统是一个基于 Electron + Vue 3 的桌面应用程序，聚焦「报销管理」与「差旅管理」。
v1.3.6 版本重点补齐差旅模块的权限控制和批量导出能力，提升数据安全性与批量处理效率。

核心能力：
- **差旅导出增强**：支持按已选或按筛选结果批量导出差旅明细。
- **多格式输出**：支持导出 Excel（CSV）与 PDF，便于归档和跨系统流转。
- **角色权限控制**：支持 admin / editor / viewer 三类角色差异化权限。
- **主进程鉴权**：关键差旅操作统一在主进程进行权限校验，避免越权调用。
- **交互一致性**：前端操作按钮按权限自动禁用并提供统一提示文案。

## 2. 技术栈 (Tech Stack)

- **桌面框架**: Electron (主进程/渲染进程架构)
- **前端框架**: Vue 3 (Composition API) + Vite
- **UI 组件库**: Element Plus
- **可视化**: ECharts
- **数据存储**: Better-SQLite3 + Electron-Store
- **自动更新**: electron-updater
- **CI/CD**: GitHub Actions

## 3. v1.3.6 关键更新

### 3.1 差旅批量导出升级
- **新增导出入口**：差旅列表支持批量导出，支持 Excel（CSV）/ PDF 两种格式。
- **导出字段完善**：覆盖 ID、报销单号、日期、金额、状态、目的地、出差时间、单据数量等核心字段。
- **导出来源灵活**：当勾选行存在时导出所选数据，未勾选时可导出当前筛选结果。
- **编码兼容性**：CSV 导出采用 UTF-8 BOM，降低中文乱码风险。

### 3.2 差旅权限控制落地
- **角色模型**：新增 admin、editor、viewer 三种角色权限映射。
- **控制开关**：新增“差旅权限控制”开关，可按环境启用或关闭权限策略。
- **链路鉴权**：主进程在查看、新建/编辑、删除、导出操作前统一校验权限。
- **前端约束**：页面内按钮、单据查看/删除、导出操作按权限禁用并提示。
