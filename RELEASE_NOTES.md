# v1.3.6 更新日志

## 差旅管理 (Travel Management)
- **Feat**: 新增差旅数据批量导出能力，支持导出为 Excel（CSV）与 PDF。
- **Feat**: 导出内容覆盖差旅核心字段（报销单号、金额、状态、目的地、出差时间、单据数量等）。
- **Feat**: 导出支持“按已选行”或“按筛选结果”两种来源，提升批量处理效率。

## 权限控制 (Permission Control)
- **Feat**: 新增差旅角色权限模型（admin / editor / viewer）。
- **Feat**: 新增差旅权限控制开关，可按配置启用或关闭权限校验。
- **Fix**: 对差旅查看、新建/编辑、删除、导出链路增加主进程权限校验，避免越权操作。

## 交互与体验 (UX)
- **Feat**: 差旅页面按钮与操作项按权限状态自动禁用并给出提示。
- **Feat**: 单据管理、查看、删除、导出等交互行为统一权限反馈文案。

## 文档与发布 (Docs & Release)
- **Docs**: 新增 `docs/erp_v1.3.6.md` 技术文档。
- **Docs**: 更新发布流程文档，补充“更新日志同步到软件发布说明”要求与发布说明模板。

## 历史版本

### v1.3.5
- **Feat**: 报销列表升级为按状态分组展示，支持分组卡片化浏览。
- **Feat**: 新增“显示已完成/隐藏已完成”开关，便于聚焦待处理单据。
- **Feat**: 新增每页条数切换（10/20），分组内独立分页。
- **Refactor**: 状态筛选项改为动态渲染，减少硬编码。
- **Fix**: 导出时自动将相对路径转换为绝对路径，兼容历史凭证路径数据。
- **Fix**: 增加文件存在性校验，缺失文件不再导致整体导出失败。
- **Feat**: 新增文件打开能力，支持在客户端直接打开导出结果或凭证文件。

### v1.3.4
- **Fix**: 修复新增报销时缺少 `receipt_no` 字段导致的提交报错问题，增加参数默认值防御。
- **Docs**: 更新技术文档至 v1.3.4，记录发布信息。
