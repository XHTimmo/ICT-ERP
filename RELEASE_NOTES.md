# v1.3.7 更新日志

## 差旅单据 (Travel Documents)
- **Feat**: 差旅单据类型统一为四类：出差申请（仅 PDF）、酒店发票、酒店水单、交通工具发票。
- **Feat**: 单据上传新增类型选择与类型提示，不同类型按规则限制可上传格式。
- **Refactor**: 兼容历史单据类型映射（如 `ticket` / `车票` / `单据`）到“交通工具发票”。

## 行程消费条例 (Itinerary Expenses)
- **Feat**: 每个行程支持维护多条消费条例（费用类型、金额、备注）。
- **Feat**: 新增消费条例管理弹窗，支持新增、删除与金额合计。
- **Feat**: 行程列表新增消费条例统计展示（条数与金额）。

## 导出增强 (Export)
- **Feat**: 差旅导出（CSV/PDF）新增“消费条例数量”“消费金额”字段。
- **Fix**: 差旅凭证 ZIP 导出适配新单据类型命名，并兼容旧字段结构。

## 文档与发布 (Docs & Release)
- **Docs**: 新增 `docs/erp_v1.3.7.md` 技术文档。
- **Docs**: 更新本次版本更新说明，供 GitHub Release 描述同步使用。

## 历史版本

### v1.3.6
- **Feat**: 差旅管理新增批量导出能力，支持 Excel（CSV）与 PDF。
- **Feat**: 新增差旅角色权限模型与权限控制开关。
- **Fix**: 差旅查看、新增/编辑、删除、导出链路增加主进程权限校验。
- **Docs**: 更新发布流程文档并补充发布说明模板。

### v1.3.5
- **Feat**: 报销列表升级为按状态分组展示，支持分组卡片化浏览。
- **Feat**: 新增“显示已完成/隐藏已完成”开关，便于聚焦待处理单据。
- **Feat**: 新增每页条数切换（10/20），分组内独立分页。
- **Refactor**: 状态筛选项改为动态渲染，减少硬编码。
- **Fix**: 导出时自动将相对路径转换为绝对路径，兼容历史凭证路径数据。
- **Fix**: 增加文件存在性校验，缺失文件不再导致整体导出失败。
- **Feat**: 新增文件打开能力，支持在客户端直接打开导出结果或凭证文件。
